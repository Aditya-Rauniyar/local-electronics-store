Built a responsive website for an existing local electronics shop to improve online presence and customer enquiries.

The website showcases products, shop details, and contact information, with direct WhatsApp enquiry support. Designed to be lightweight, fast, and easy to use on mobile devices.

## Live Deployment
https://santosh-electronics-in.vercel.app/

## Tech Used
- React (Vite)
- Tailwind CSS
- Framer Motion
- Vercel (deployment)

## Purpose
- Display available products
- Share shop location and contact details
- Enable quick customer enquiries via WhatsApp
- Provide a simple digital presence for a local business
